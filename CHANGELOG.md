# Changelog / История изменений

## 2026

### Версия 1.3 (13 апреля 2026)
- Создан EXE с графическим интерфейсом (GUI)
- Исправлена загрузка шрифта и масок
- Добавлен блок для текста события между годом и картинкой

### Версия 1.2 (13 апреля 2026)
- Исправлено центрирование текста на каждой стороне
- Добавлены функции wrap_text() и draw_centered_text()

### Версия 1.1 (13 апреля 2026)
- Центрирование текста

### Версия 1.0 (2026)
- Создан four_sided_display.py - генератор изображений
- Загрузка событий с Wikipedia API
- Генерация PNG 2080x1872 (4 стороны по 520px)

## GUI версия

- historical_display_gui.py - полноценный GUI с выбором даты, событий, редактированием текста и загрузкой изображений
- **Важно**: На Python 3.14 может вылетать из-за бага GC

## Файлы

- `HistoricalDisplay.exe` - Готовый EXE с GUI
- `four_sided_display.py` - CLI версия
- `generate_cli.py` - Упрощенная CLI версия

## Запуск

```bash
# EXE
HistoricalDisplay.exe

# CLI
python four_sided_display.py 4 13
```
