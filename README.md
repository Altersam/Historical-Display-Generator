# Historical Display Generator

Приложение для создания информационных плакатов с историческими событиями для 4-стороннего LED-экрана.

## Возможности

- Графический интерфейс (GUI)
- Загрузка событий с Wikipedia API
- Редактирование года и текста событий
- Выбор изображения из списка
- Генерация изображения 2080x1872 пикселей
- Автоматический подбор размера шрифта

## Использование

1. Выберите день и месяц
2. Нажмите "Загрузить" для загрузки событий
3. Для каждой стороны выберите событие из списка
4. При необходимости отредактируйте год и текст
5. Нажмите "Собрать", затем "Сохранить"
